这里是 Daisy 2024年10月的每日总结。

这篇博客存在的目的是想让自己养成记录与总结的习惯，希望能够通过写作这一途径对自己一天的学习生活做一些简单的总结与反思。

2024.10.24

学完了西瓜书第四章神经网络的内容，准备向 SVM 推进。

由于模式识别上机错过了与学长约定的交流时间，先自己琢磨琢磨解决方案吧。

晚上开始喉咙有些不舒服，感觉可能是感冒来临的前兆，先喝了杯板蓝根，再把自己裹严实了，希望能缓解整整。

由于身体不舒服，就没有做脑力活，简单看了一下 cloudflare 的用法，注册了一个一年期的域名，准备这两天动手实施新的图床方案。

明天加油！

2024.10.23

今天完成了模式识别的第一次大作业，用python 实现了 Fisher 在两个数据集上的线性判别分析，基本没调库，纯手搓，感觉做的还挺完整像样的，不错不错。

工作方面问题层出，明天找学长交流求教。

晚上的三千米比昨天又快了一点，慢慢进步吧，罗马不是一日建成的，体能也不是一日积累的。

明天加油！

2024.10.22

今天主要是完成了课后作业，工作方面出现了一些奇怪的问题，心里又开始有些小慌。但是，如果我还怀揣着深造的念头，就应该沉着应对眼前的困难
，冷静的思考解决方案，锻炼自己独立思考的能力，方法总是比困难多的！

机器学习逻辑回归作业完成，还挺搞笑的，找了半天的 bug 竟然是因为自己在求完损失梯度的时候没把他赋值传回去，导致整个训练过程梯度全是零哈哈哈，
虽然很搞笑，但还是警醒我要认真一点。

今天也跑了三公里，配速比昨天快了一分钟，昨天简直是在操场爬行（bushi），继续加油。

这两天ddl比较多，也有好几次上机，希望自己能够合理安排时间，争取多做些事情。

继续加油吧！

2024.10.21

今天没有什么实际进度，主要是和学长交流了一下，明确了下一步动作，得到肯定还是一个令人开心的事情。

晚上跑了三千米，配速不快，跑的也不累，明天也要跑，中断几天再跑明显退步了。

比较烦人的是，今天发现我托管在 github 上的博客图片的 cdn 加速挂掉了，导致我近一半的博客图片无法访问，很烦，但暂时也不想花时间管了，后续一个可能的解决方案是切换图床方案为 cloudflare + X，在此标记一下，等周内闲下来再搞。

2024.10.20

今天稍微有点划水，机器学习作业出了一些问题，notebook debug又较为麻烦，进度大约 2 / 3，争取明天完结。

工作方面，一些可行性验证明天上午推进。

以及重要的事情说三遍：kaggle! kaggle! kaggle!

最后，天气越来越冷了，得加几件衣服，避免着凉~

明天又是新的一周，继续加油吧！

2024.10.19

高强度作业15小时，但是不知道为啥好像还是没有重大进展。

上午写完了 Vision + X 的论文学习博客，总算了结心头一患， debug 一个下午，晚上敲了一半决策树笔记，顺便把去年暑假做的数据结构笔记更新在博客上。

完成了知识工程的作业，初探机器学习作业，明天写完。

整合了一些问题，周一和学长交流。

明天加油！

2024.10.18

时间过得有点快。

晚上画了一下图整理了思路，遇到了许多预料以外的问题，但人的能力正是在不断发现并解决问题的过程中成长的，不要气馁。

终于迎来了两天的周末，乘着不断电晚上小熬一会，希望有新进展。

今天没什么时间写博客，就暂时不更新机器学习笔记了，明天一切如常。

继续加油！

2024.10.17

模型大压缩，但是跑不动，明天换一个思路徐图，晚上好不容易没报错结果测 metric 的时候 output 直接爆 nan 了。

西瓜书的决策树部分内容看完了，线性模型部分博客更新完成。

晚上小小打了两把游戏放松了一下，又调了一个多小时模型就断电了，反正明天没早八，把 Vision + X 看完再上床吧。

遇到小挫折，也得继续加油，办法总比困难多。

明天加油！

2024.10.16

晚上听学长的讲座，感叹时光飞逝，自己已经是大三老登了。

西瓜书决策树章节看了一半左右，笔记也有做，决定每天用中午时间更新博客相关笔记，提高时间利用率。

明天继续加油！

2024.10.15

英语六级备考正式开始，每日五十词启动。

西瓜书读完第三章，准备向第四章推进，笔记每天会更新一部分到博客上（敲 latex 公式太费时间了）。

工作开始推进，目前已经对如何 scale 模型尺度有了初步想法，也将持续推进。

Vision + X 文章还剩最后一部分-多模态生成模型研究，明天会看完。

写博客感想：时间成本比较大，希望收获可以更大。

明天加油！

2024.10.14

将博客中的图片在 picx 中进行压缩与托管，脱离了管理博客图片文件的苦恼，很是方便。

读完了西瓜书第三章线性模型，手推公式并记笔记，感觉学到了点东西，然而在把笔记整理成 md 文档的时候被 mathjax 公式折磨的不成样子。

下午和带我的学长聊了一下规划，确定了训练时间慢的改进方案，并决定年末抓紧时间推进工作，希望顺利。

最后检讨一下，不太应该在晚上花三个小时又敲又改 mathjax 公式，总是本地渲染得很明白到了云端就不渲染了，有点和它杠上了。

明天加油！

2024.10.13

ddl暂时赶完了，今天实在没有状态，刚上完五天课，马上又上五天课了。

为什么课总是那么多，感觉时间不够用，不休息又很累。

明天找学长推进工作，得加油了。

2024.10.12

和学长联系了一下，还是决定无论如何先将论文推进完，希望自己能争点气吧。

机器学习作业代码粗略写了一下，有些bug，明天再看。

周六还要上三节课，是真的累了，晚上打会CS放松了一下，明天整理好心情继续加油吧。

2024.10.11

终于做完了机器学习的书面作业，禁不住感叹怎么会有那么多题…

拜一天满课加作业所赐，手头上做的小玩意没什么进展，明儿还要上课，也不知道啥时候能搞完。

明天加油吧。

(至少让我把 CLIP 复现完啊 T_T)

2024.10.10

机器学习作业是真的多，还好没有等ddl当天赶，不然铁定来不及…

今天继续复现 CLIP ，模型的架构是搭好了，具体的可行性验证有待明日开展，其实感觉代码里有点小问题，但说不出来hh，目前看来，张量流在模型中的流动还算符合预期，明天争取解决。

读论文的博客撰写工作也有待推进，以及 llm 的学习与实践也得提上日程。总有一种事追着人跑的感觉。

今天没什么新鲜事，知识工程的论文分享由我进行汇报，也算解决了一桩事，也跑了个三千米，比昨天快点，但还是缺乏锻炼的弱鸡一个，路漫漫其修远兮！

明天加油，望一切顺利。

2024.10.9

今天并没有推进 KuiperInfer 的第三课，重新思考了一下，深度学习框架较为底层，还是细嚼慢咽比较好。

今天完成了 CLIP 模型的文本编码器和图像编码器，文本编码器还需要和 logits 对齐，这是明天的工作。

Transformer 的架构蛮多模型在用的，今天发现文本编码器的编码意外流畅许多，虽然还是有想不起来的地方，但也不着急，步步为营，稳扎稳打，才能行稳致远，学以致用。

对了，整了一下午知识工程课程PPT，读一篇近二十页的综述？感觉更像科普文。还和朋友跑了三千米，蛮记录下了。

明天加油！

2024.10.8

第二课进行中，很遗憾没能在当日实现Tensor::Padding函数的编写，希望明天尽快解决后进入下一课。

这是我第一次接触中大型 C++ 项目，顿感之前写的算法题都是小打小闹，学到了很多新奇的玩意，比如 C/C++ 项目在运行前必须编译链接，也小小地了解了一下 CMakeFile.txt 的大致内容和作用（跨平台编写makefile）。

估计这个项目没有十天完不成，先沉下心慢慢做吧。

至于读论文和其他的内容，还是处于搁置状态，毕竟今天一天满课，能够省下的时间屈指可数啊，悲伤。

个人感觉这个 Kuiperinfer 项目可以和别的内容并行，这几天打算同步复现一下 Clip，还是希望每天至少挤出来一个半小时啊。

对了，千万别忘了即将截止的机器学习作业和知识工程汇报，生活就得如此充实😀。

明天加油吧！

2024.10.7

完成了环境配置，大致明白了项目的实现流程，接下来就是跟着 Instruction 一步一步实现了。

国庆的最后一天，还是有点怠惰，明天开始要收心了，别忘了自己的规划！

2024.10.6

今天开始做用 C++ 实现深度学习推理框架的项目，大致明白了 Docker 的用途和使用方法，其技术细节看看后续有没有机会接触学习一下。

VSCode 不知道为什么连不上 Docker 的 ssh，要用宿主机的 ip 连接，对于传输协议之类的知识并不是很熟悉，以后有机会也研究一下。

明天开始要正式推进项目主体了，希望这一周能够做完这个项目。

加油！

2024.10.5

关于为什么3号、4号没有写博客，因为当时我正在甘肃天水进行大学生特种兵两天体验，实在没有写博客的余裕。

这也是我第一次和朋友出远门，平时都是在所在市的市内随便逛逛，小搓几顿，也算是值得纪念的首次出游。

关于体验呢，其实也还不错，有很多有意思的地方，也有一些小插曲，相信这次体验能给我下次的出游增添许多底气，这两天也打算写一篇博客记录一下几经波折的天水二日游。

距离大学生活结束只剩不到两年时间，也许也可以开始憧憬一下毕业旅行了哈哈哈。

P.S. 这次旅行同样也是火车硬座的初次体验，我的评价是，在硬座上能睡着的，也是神人了！

2024.10.2

从零实现 Transformer 项目已经颇有几分模样，心中还是有些成就感的。

博客的相关部分还没有完成，估计得到10.5以后才能完善，无论如何，先享受假期吧。

明早六点五十分就要出发，目标天水，一切顺利！

2024.10.1

今天算是完整实现了 Transformer 的整体架构，准备在这个架构的基础之上实现一个简单的翻译器，感觉在这个过程还是学到了很多，也是真正地做了一个可以称之为项目的工作。

尽量在国庆出游前完成这个项目，然后开开心心玩两天。

下一个项目已经决定好了，用 C++ 实现一个深度学习推理库。多模态论文阅读也得提上日程了。

明天继续加油。
